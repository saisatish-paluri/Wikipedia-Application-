Wikipedia Search Application

This is a small web application that lets users search for information on Wikipedia. When a user types a keyword and clicks search, the app fetches results from the Wikipedia API and shows the title, a short description, and a link to the full article.

Features

  1. Search anything on Wikipedia
  2.Shows result titles, descriptions, and article links
  3.Works with the Wikipedia API
  4.Simple and lightweight
  5.Easy to understand code structure

Technologies Used

  1)HTML
  2)CSS
  3)JavaScript
  4)Wikipedia API (for fetching search results)


How to Run

  1)Download or clone the project.
  2)Open the index.html file in any web browser.
  3)Enter a search term and check the results.
